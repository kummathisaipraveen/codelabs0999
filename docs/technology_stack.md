# Technology Stack: CodeCoach

CodeCoach is built with a modern, high-performance stack designed for scalability, security, and a premium developer experience.

## 🎨 Frontend (React / Vite)
- **React**: The core library for building a dynamic, component-based user interface.
- **Vite**: A lightning-fast build tool that provides a smooth development experience and highly optimized production bundles.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development and precise styling.
- **Shadcn UI**: A collection of high-quality, accessible UI components built with Radix UI primitives.
- **Framer Motion**: A powerful library for creating smooth, production-ready animations and transitions.
- **Lucide React**: For beautiful, consistent, and customizable vector icons.

## ⚙️ Backend (Python / FastAPI)
- **FastAPI**: A modern, high-performance web framework for Python 3.8+ based on standard Python type hints.
- **Python**: The primary language for all server-side logic, AI orchestration, and data analysis.
- **Uvicorn**: A lightning-fast ASGI server implementation for running the FastAPI application.
- **Python-jose**: For secure JWT (JSON Web Token) decoding and session validation.

## 🗄️ Database & Security (Supabase)
- **Supabase**: An open-source Firebase alternative that provides a suite of tools for rapid building and scaling.
- **PostgreSQL**: The world's most advanced open-source relational database for persistent storage.
- **Row Level Security (RLS)**: Fine-grained access control enforced directly at the database level to ensure 100% data isolation.
- **Supabase Auth**: A secure identity management system for handling student and teacher accounts.

## 🧠 Specialized Services & AI
- **Pyodide**: A Python distribution for the browser (WebAssembly) that allows student code to execute locally with zero latency.
- **Google Gemini AI**: A state-of-the-art Large Language Model (LLM) that powers the Socratic tutor and progressive hint system.
- **NetworkX**: A comprehensive Python library for analyzing complex networks and graphs, used to manage the adaptive learning ontology.
- **React Query**: For efficient, high-performance data fetching, caching, and state synchronization with the backend.
